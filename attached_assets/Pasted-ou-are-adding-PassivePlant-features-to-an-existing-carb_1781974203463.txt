ou are adding "PassivePlant" features to an existing carbon footprint app that already has CarbonSwap and AutoCarbon Null. PassivePlant makes your PHONE USAGE automatically plant trees. User uses phone normally, trees grow in background. Zero effort.

## GOAL
Add PassivePlant phone-tracking + auto-tree-planting to existing app WITHOUT breaking current functionality. Combine with existing CarbonSwap (coins) and AutoCarbon Null (AI offsets).

## DATABASE CHANGES (Add These Tables)

Create these NEW tables in existing PostgreSQL:

1. phone_usage:
   id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
   user_id UUID REFERENCES users(id) ON DELETE CASCADE,
   usage_date DATE NOT NULL,
   screen_time_hours DECIMAL DEFAULT 0,
   app_usage_hours DECIMAL DEFAULT 0,
   battery_drain_mah DECIMAL DEFAULT 0,
   co2_kg DECIMAL DEFAULT 0, -- Calculated: 1 hour screen = 0.01kg CO₂
   created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

2. tree_plants:
   id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
   user_id UUID REFERENCES users(id) ON DELETE CASCADE,
   plant_date DATE NOT NULL,
   trees_planted INTEGER DEFAULT 0,
   co2_offset_kg DECIMAL DEFAULT 0, -- 1 tree = 0.01kg CO₂ captured/year
   location TEXT, -- Forest location (e.g., "Kerala Forest")
   project_name TEXT, -- "Plant 1000 Trees Initiative"
   is_verified BOOLEAN DEFAULT false,
   blockchain_tx_id TEXT, -- Polygon transaction ID
   photo_url TEXT, -- Tree growth photo
   created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

3. tree_plant_donations:
   id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
   user_id UUID REFERENCES users(id) ON DELETE CASCADE,
   tree_plant_id UUID REFERENCES tree_plants(id) ON DELETE CASCADE,
   screen_time_hours DECIMAL,
   trees_earned INTEGER,
   created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

4. passive_rewards:
   id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
   user_id UUID REFERENCES users(id) ON DELETE CASCADE,
   reward_type TEXT, -- 'tree', 'coin', 'discount'
   reward_value DECIMAL, -- Number of trees/coins/discount ₹
   reward_date DATE,
   is_claimed BOOLEAN DEFAULT false,
   created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

Add indexes for: user_id in all tables, usage_date in phone_usage, plant_date in tree_plants.

## BACKEND API ENDPOINTS (Add to Your Express App)

Add these NEW routes:

1. POST /api/phone/track - Track phone usage (called by mobile app)
   - Input: { screen_time_hours, app_usage_hours, battery_drain_mah }
   - Logic:
     • CO₂ calculation: screen_time_hours * 0.01kg + app_usage_hours * 0.005kg
     • Create phone_usage record
     • Auto-earn trees: 1 tree per 10kg CO₂
     • trees_earned = co2_kg / 10
     • If trees_earned >= 1:
       - Create tree_plant record (trees_planted: floor(trees_earned))
       - Create tree_plant_donation record
       - Auto-add to eco_wallets.coin_balance += trees_earned * 10 (1 tree = 10 coins)
   - Return: { success: true, co2_kg: 0.15, trees_earned: 0.015, total_trees: 15 }

2. GET /api/phone/usage - Get phone usage summary
   - Input: { user_id, period: 'week' | 'month' }
   - Logic:
     • Sum phone_usage for period
     • total_screen_hours, total_app_hours, total_co2_kg
     • total_trees_planted (from tree_plants)
   - Return: { screen_hours: 120, app_hours: 80, co2_kg: 1.2, trees_planted: 12 }

3. GET /api/trees - Get user's planted trees
   - Logic:
     • Get tree_plants for user
     • Include: location, project_name, trees_planted, photo_url, is_verified
   - Return: [{ plant_date, trees_planted: 10, location: 'Kerala Forest', photo_url: 'URL' }]

4. POST /api/trees/plant - Auto-plant trees (manual trigger or auto)
   - Input: { co2_kg, project_name, location }
   - Logic:
     • trees_to_plant = co2_kg / 0.01 (1 tree = 0.01kg CO₂/year)
     • Cost: ₹5 per tree (₹50 for 10 trees)
     • Auto-pay from eco_wallets.money_balance OR subscription
     • Create tree_plant record
     • Generate blockchain transaction (Polygon)
     • Upload photo (tree growth)
     • Set is_verified = true
   - Return: { success: true, trees_planted: 10, cost: 50, blockchain_tx: "TX_ID" }

5. GET /api/trees/map - Get live tree map
   - Logic:
     • Get all tree_plants
     • Group by location
     • Return: [{ location, total_trees: 160, co2_offset: 1.6kg }]
   - Return: [{ location: 'Kerala Forest', trees: 160, co2: 1.6 }, { location: 'Tamil Nadu', trees: 80, co2: 0.8 }]

6. POST /api/rewards/claim - Claim passive rewards
   - Input: { reward_id }
   - Logic:
     • Get reward (type: 'tree', 'coin', 'discount')
     • If type = 'tree': Create tree_plant
     • If type = 'coin': Add to eco_wallets.coin_balance
     • If type = 'discount': Generate discount coupon
     • Set is_claimed = true
   - Return: { success: true, reward_type: 'tree', reward_value: 5 }
