You are adding "PassivePlant" features to an existing carbon footprint app that already has CarbonSwap and AutoCarbon Null. PassivePlant makes your PHONE USAGE automatically plant trees. User uses phone normally, trees grow in background. Zero effort.

## GOAL
Add PassivePlant phone-tracking + auto-tree-planting to existing app WITHOUT breaking current functionality. Combine with existing CarbonSwap (coins) and AutoCarbon Null (AI offsets).

## DATABASE CHANGES (Add These Tables)

1. phone_usage:

     • CO₂ calculation: screen_time_hours * 0.01kg + app_usage_hours * 0.005kg
     • Create phone_usage record
     • Auto-earn trees: 1 tree per 10kg CO₂

     • If trees_earned >= 1:
       - Create tree_plant record (trees_planted: floor(trees_earned))
       - Create tree_plant_donation record
       - Auto-add to eco_wallets.coin_balance += trees_earned * 10 (1 tree = 10 coins)
   - Return: { success: true, co2_kg: 0.15, trees_earned: 0.015, total_trees: 15 }

   - Input: { user_id, period: 'week' | 'month' }

     • total_trees_planted (from tree_plants)
   - Return: { screen_hours: 120, app_hours: 80, co2_kg: 1.2, trees_planted: 12 }

   - Logic:
     • Get tree_plants for user

   - Input: { co2_kg, project_name, location }

5. GET /api/trees/map - Get live tree map

### 1. Phone Usage Tracker (Background Service)

```javascript
// mobile/src/services/PhoneUsageTracker.js

import { DeviceEventEmitter } from 'react-native';
import api from '../services/api';

class PhoneUsageTracker {
  constructor() {
    this.screenTime = 0;
    this.appUsage = {};
    this.batteryDrain = 0;
    this.startTracking();
  }

  startTracking() {
    // Track screen on/off
    DeviceEventEmitter.addListener('screenOn', () => {
      this.screenStartTime = Date.now();
    });

    DeviceEventEmitter.addListener('screenOff', () => {
      const screenTime = (Date.now() - this.screenStartTime) / 1000 / 60 / 60; // hours
      this.screenTime += screenTime;
      this.submitUsage();
    });

    // Track app usage
    DeviceEventEmitter.addListener('appOpen', (appName) => {
      this.appStartTime = { app: appName, time: Date.now() };
    });

    DeviceEventEmitter.addListener('appClose', (appName) => {
      const appTime = (Date.now() - this.appStartTime.time) / 1000 / 60 / 60;
      this.appUsage[appName] = this.appUsage[appName] || 0;
      this.appUsage[appName] += appTime;
      this.submitUsage();
    });

    // Track battery drain
    DeviceEventEmitter.addListener('batteryChange', (mAh) => {
      this.batteryDrain += mAh;
    });

    // Submit usage every hour
    setInterval(() => this.submitUsage(), 60 * 60 * 1000);
  }

  submitUsage() {
    const screenTimeHours = this.screenTime;
    const appUsageHours = Object.values(this.appUsage).reduce((a, b) => a + b, 0);
    const batteryDrainMah = this.batteryDrain;

    // Calculate CO₂
    const co2Kg = screenTimeHours * 0.01 + appUsageHours * 0.005;

    // Submit to API
    api.post('/api/phone/track', {
      screen_time_hours: screenTimeHours,
      app_usage_hours: appUsageHours,
      battery_drain_mah: batteryDrainMah
    });

    // Reset
    this.screenTime = 0;
    this.appUsage = {};
    this.batteryDrain = 0;
  }
}

export default new PhoneUsageTracker();
```
